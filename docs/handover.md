# Handover: Jakob to Team Member

## Goal
Ensure any team member can run sales to onboarding without loss of context.

## Handover Package (Per Customer)
- Company ID and portfolio ID
- Current CRM status and next action
- Missive thread link
- Investor profile sliders (data granularity, action frequency, risk appetite, decision logic)
- Output format and cadence
- Latest delivery asset link
- Add as ingestion an inbox system where we forward mail to a AI knowledge based on different newsletters subscribed

## Checklist
1. Confirm `customer_basic_info.status` and `action_status` are current.
2. Verify portfolio assets exist in `portfolio_assets`.
3. Validate investor profile fields in `investor_profiles`.
4. Record handover note (Loom or voice) and post to Missive/Telegram.
5. Set next scheduled check date.

## Red Flags
- Missing email or portfolio data
- No decision logic set (defaults to 3)
- Product status stuck in "to review" for > 5 days
